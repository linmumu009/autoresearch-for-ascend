# v0.2.2 Update - Converted MindSpeed HF Evaluation

Date: 2026-07-02

## What Changed

- Added `ascend_autoresearch/evaluate_hf.py`, an eval-only script that reuses
  the fixed `prepare.py` validation path.
- Added `framework_adapters/mindspeed_llm/convert_mcore_to_hf.sh` for
  MindSpeed/MCore to Hugging Face conversion.
- Documented the first same-surface raw validation comparison.

## Result

The MindSpeed checkpoint from the 6-step validation smoke was converted to HF
format and evaluated with the same raw next-token validation surface as the HF
thin loop.

| Model | Raw HF Val Loss | Raw HF Val PPL |
| --- | ---: | ---: |
| Base Qwen3-0.6B | 14.977717 | 3196979.536 |
| Converted MindSpeed 6-step checkpoint | 14.963873 | 3153024.804 |

## Interpretation

The improvement is small, as expected from only 6 micro steps. The meaningful
milestone is the closed evaluation loop: MindSpeed training output can now be
converted and judged by the same raw HF validation code.
