# v0.3.0 Update - MindSpeed Autoresearch Runner

Date: 2026-07-03

## What Changed

- Added a MindSpeed-LLM candidate runner.
- Added a baseline 6-step candidate env file.
- Made the MindSpeed training script honor `LR` so learning-rate search is
  actually possible.

## Candidate Loop

The runner executes:

1. MindSpeed-LLM training,
2. MCore to Hugging Face conversion,
3. fixed raw HF validation with `evaluate_hf.py`,
4. TSV metric recording.

Default output:

```text
/workspace/runs/mindspeed_llm/results.tsv
/workspace/runs/mindspeed_llm/<run_name>/
```

## Why

Previous versions proved the pieces independently. This version makes them a
single repeatable experiment unit, which is the shape needed for real
autoresearch.
