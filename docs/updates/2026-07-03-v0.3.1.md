# v0.3.1 Update - First MindSpeed LR Search

Date: 2026-07-03

## What Changed

- Added three candidate env files for the first MindSpeed-LLM learning-rate
  search.
- Ran each candidate with the same train -> convert -> HF eval -> record loop.
- Updated documentation with the first search result.

## Results

| Candidate | LR | Raw HF Val Loss | MindSpeed Valid Loss | Last Train Loss |
| --- | ---: | ---: | ---: | ---: |
| baseline | `1.25e-6` | `14.962966` | `0.612093` | `0.844955` |
| low | `6.25e-7` | `14.975369` | `0.615938` | `0.847379` |
| high | `2.5e-6` | `14.927566` | `0.590943` | `0.826132` |
| higher | `5.0e-6` | `14.819130` | `0.532507` | `0.772222` |

## Interpretation

The search signal is clean in this small budget: lower LR got worse, higher LR
got better, and `5.0e-6` is the current best. The next pass should explore
around `7.5e-6` and `1.0e-5` before moving to longer training budgets.
