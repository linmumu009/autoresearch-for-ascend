# v0.3.10 - 9e-5 Short-Budget Probe

Date: 2026-07-03

## What Changed

- Added `framework_adapters/mindspeed_llm/candidates/lr_9em5_6step.env`.

## Why

The v0.3.9 run showed that `LR=8.0e-5` still improved the 6-step boundary, but
only slightly. This version probes `9.0e-5` to continue bracketing the
short-budget optimum.

Probe `1.0e-4` on the 6-step boundary. If it worsens, bracket the short-budget
optimum between `9.0e-5` and `1.0e-4`.
