# v0.3.2 - Extended MindSpeed LR Range Search

Date: 2026-07-03

## What Changed

- Added `framework_adapters/mindspeed_llm/candidates/lr_7p5em6_6step.env`.
- Added `framework_adapters/mindspeed_llm/candidates/lr_1em5_6step.env`.
- Updated the MindSpeed-LLM adapter notes, framework evaluation, changelog, and
  main README with the new candidate results.

## Why

The v0.3.1 search showed that `LR=5.0e-6` was better than `2.5e-6` and the
runner baseline under the same 6-step budget. This version checks whether that
trend continues or starts to over-shoot.

## Results

| Candidate | LR | Raw HF Val Loss | Raw HF Val PPL | MindSpeed Valid Loss | Last Train Loss |
| --- | ---: | ---: | ---: | ---: | ---: |
| `lr_7p5em6_6step.env` | `7.5e-6` | 14.622040 | 2240120.089 | 0.448289 | 0.693946 |
| `lr_1em5_6step.env` | `1.0e-5` | 14.503275 | 1989263.400 | 0.408763 | 0.643258 |

## Decision

Keep `lr_1em5_6step.env` as the current best 6-step MindSpeed candidate.

Compared with the runner baseline raw HF val_loss `14.962966`, it improved by
`0.459691`. Compared with base Qwen3-0.6B raw HF val_loss `14.977717`, it
improved by `0.474442`.

## Next Step

Probe the over-shoot boundary above `1.0e-5`, for example `1.5e-5` and
`2.0e-5`, before spending a longer training budget.
