# v0.3.9 - 8e-5 Short-Budget Probe

Date: 2026-07-03

## What Changed

- Added `framework_adapters/mindspeed_llm/candidates/lr_8em5_6step.env`.
- Updated English and Chinese README files, framework evaluation notes,
  changelog, and MindSpeed-LLM adapter notes with the new result.

## Why

The v0.3.8 run showed that `LR=7.0e-5` still improved the 6-step boundary. This
version probes `8.0e-5` to see whether the short-budget run starts to over-shoot.

## Results

| Candidate | LR | Steps | Raw HF Val Loss | Raw HF Val PPL | MindSpeed Valid Loss | Last Train Loss |
| --- | ---: | ---: | ---: | ---: | ---: | ---: |
| `lr_8em5_6step.env` | `8.0e-5` | 6 | 12.434883 | 251420.643 | 0.372704 | 0.590512 |

## Decision

Keep `lr_8em5_6step.env` as the best observed candidate overall.

The 6-step run still improved, but the gain over `7.0e-5` is small. This is a
sign that the short-budget boundary may be close.

## Next Step

Probe `9.0e-5` on the 6-step boundary. If it worsens, bracket the short-budget
optimum between `8.0e-5` and `9.0e-5`.
